Les Briques de Base

Concept A Variables & Entrées

Une variable est une boîte pour ranger une information. input() permet de poser une question à l'utilisateur.

# On crée une boîte nommée 'joueur'
# On y range la réponse de l'utilisateur
joueur = input("Quel est ton nom ? ")

# On utilise le contenu de la boîte
print(f"Enchanté, {joueur} !")

Concept B Les Conditions (Si... Sinon)

Le programme doit prendre des décisions. En Python, on utilise if (si), elif (sinon si) et else (sinon).

age = 12

if age < 10:
    print("Tu es un enfant.")
elif age < 18:
    print("Tu es un ado.")
else:
    print("Tu es un adulte.")

Le Projet Final : Jeu de Devinette

L'Objectif

L'ordinateur choisit un nombre secret entre 1 et 100. Vous devez le deviner. À chaque essai, il vous dira si c'est "plus" ou "moins". Ce projet est un classique du livre "Programmer en s'amusant" (Page 82).

Le Code Complet :

import random # On importe la boîte à outils du hasard

print("--- JEU DE DEVINETTE ---")
nombre_secret = random.randint(1, 100)
essais = 0

print("Je pense à un nombre entre 1 et 100.")

while True:
    # On demande un nombre (et on le convertit en entier)
    proposition = int(input("Ta proposition : "))
    essais = essais + 1

    if proposition == nombre_secret:
        print(f"Gagné ! Tu as trouvé en {essais} coups !")
        break # On sort de la boucle, le jeu est fini
    elif proposition < nombre_secret:
        print("C'est plus grand ⬆️")
    else:
        print("C'est plus petit ⬇️")
Analyse du code :
  • import random : Apporte les fonctions de hasard (Chapitre 10 de Python pour les Nuls).
  • while True : Crée une boucle infinie qui ne s'arrête que si on gagne (break).
  • int(...) : Convertit le texte tapé par le joueur en nombre pour pouvoir le comparer.

Les Outils Essentiels

print() Afficher à l'écran

La fonction print() est votre façon de "parler" à l'utilisateur. Elle affiche du texte (entouré de guillemets) ou le contenu d'une variable.

message = "Bonjour le monde !"
print(message) 
print("Mon âge est :", 30) # Affiche plusieurs choses à la suite
# Résultat : Bonjour le monde !
# Résultat : Mon âge est : 30

Le Terminal (CMD ou PowerShell)

Le **Terminal** (ou Command Prompt sur Windows, ou Terminal sur Mac/Linux) est l'endroit où vous pouvez lancer vos scripts Python directement. C'est l'interface texte de votre ordinateur.

Comment exécuter votre jeu de devinette :

  • 1. Sauvegardez votre code dans un fichier, par exemple jeu_devinette.py.
  • 2. Ouvrez votre Terminal (CMD).
  • 3. Allez dans le dossier où vous avez sauvegardé le fichier (commande cd votre_dossier).
  • 4. Lancez le script :
python jeu_devinette.py

Installer des "Boîtes à outils" avec pip et pyLab

Qu'est-ce que pip ?

pip est un outil magique qui permet d'installer des bibliothèques (des "boîtes à outils" supplémentaires) créées par d'autres programmeurs. Par exemple, si vous voulez dessiner des graphiques, vous avez besoin de la bibliothèque matplotlib.

Exemple d'utilisation de pip dans le Terminal :

pip install matplotlib

Et pyLab ?

pyLab est en fait un module à l'intérieur de la bibliothèque matplotlib, très populaire dans le domaine scientifique. Il simplifie les commandes pour faire des calculs et afficher des graphiques rapidement. C'est votre porte d'entrée vers l'analyse de données et l'IA !

# Exemple d'utilisation dans un script Python (après 'pip install matplotlib')
import pylab as plt

plt.plot([1, 2, 3], [1, 4, 9]) # Dessine une ligne
plt.title("Mon premier graphique")
plt.show()

Le Langage : Quand utiliser quel symbole ?

"" ou '' Les Guillemets

Les guillemets simples (' ') ou doubles (" ") servent à délimiter du **texte pur** (appelé `string`). Python n'essaiera pas d'interpréter ce qui se trouve à l'intérieur comme du code. L'important est d'utiliser la même paire :

# Correct :
nom = "Alice" 
phrase = 'J\'aime Python' 

# Erreur (mixte) :
# mauvaise_phrase = "Je suis " Bob'

() Les Parenthèses

Elles ont deux rôles principaux :

  • **Appeler une fonction :** Quand on demande à une fonction de travailler (print(), input(), random.randint()).
  • **Définir l'ordre de priorité :** Comme en mathématiques (ex: resultat = (10 + 5) * 2).
  • **Créer un `tuple` :** Une liste qui ne peut pas être modifiée après sa création.

[] et {} Crochets et Accolades

  • **Crochets `[]` :** Utilisés pour les **Listes** (collections d'éléments modifiables) et pour accéder à un élément précis dans une liste (ma_liste[0]).
  • **Accolades `{}` :** Utilisées pour les **Dictionnaires** (pour stocker des informations sous forme de paires clé:valeur) et les **Ensembles**.
ma_liste = ["pomme", "banane"] # Crochets
mon_profil = {"nom": "Bob", "age": 25} # Accolades

Raccourcis Clavier : Gagnez du Temps !

Ces raccourcis fonctionnent dans la plupart des éditeurs de code comme **IDLE**, Visual Studio Code, ou d'autres.

Commandes Essentielles

  • Ctrl/Cmd + S : **Sauvegarder** le fichier.
  • F5 (dans IDLE) : **Exécuter** le script.
  • Ctrl/Cmd + C : **Copier** le texte sélectionné.
  • Ctrl/Cmd + V : **Coller** le texte.

Productivité & Débogage

  • Ctrl/Cmd + Z : **Annuler** la dernière action.
  • Ctrl/Cmd + F : **Rechercher** un mot dans le fichier.
  • Ctrl/Cmd + / : **Commenter/Décommenter** une ligne de code (très pratique pour désactiver une ligne temporairement).
  • Tab / Shift + Tab : **Indenter** (décaler) le code (essentiel en Python).
Astuce Indentation : En Python, l'**indentation** (les espaces au début d'une ligne) est cruciale pour définir les blocs de code (if, else, while). Utilisez la touche Tab ou **4 espaces** pour indenter.

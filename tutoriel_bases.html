Le Langage : Quand utiliser quel symbole ?

"" / '' Les Guillemets (Délimiteur de Texte)

Délimitent le **texte pur** (type `string`). Python n'interprète pas ce qui est à l'intérieur comme du code. On peut utiliser des guillemets simples ou doubles, mais il faut être cohérent.

Exemple Ligne par Ligne : 
# Assignation : La valeur (texte) "Ceci est une chaîne de caractères." est stockée dans la variable 'message'
message = "Ceci est une chaîne de caractères."

() Les Parenthèses (Appel de Fonction / Ordre)

Elles ont deux fonctions principales : l'appel de fonction et la définition de la priorité des opérations.

Exemple Ligne par Ligne : 
print("Salut")              # Exécute la fonction 'print' en lui donnant "Salut" comme argument (texte à afficher)
resultat = (10 + 5) * 2     # Les parenthèses forcent l'opération '10 + 5' (15) à être exécutée avant la multiplication par 2

[] et {} Crochets et Accolades (Collections)

Elles sont utilisées pour créer et manipuler les collections complexes de données.

Exemple Ligne par Ligne : 
ma_liste = [10, 20, 30]           # Les crochets créent une LISTE de trois nombres
print(ma_liste[0])                # Les crochets sont utilisés pour ACCÉDER au premier élément de la liste (index 0)

mon_dict = {"nom": "Paul", "age": 30} # Les accolades créent un DICTIONNAIRE de paires Clé:Valeur
print(mon_dict["nom"])            # Les crochets sont utilisés pour ACCÉDER à la valeur associée à la clé "nom"

Les Briques de Base & Prise de Décision

A Variables & Entrées (input())

Le signe **`=`** est l'opérateur d'assignation. Il place la valeur de droite dans la variable nommée à gauche.

Exemple Ligne par Ligne : 
reponse = 42                                     # Assignation : La valeur numérique 42 est stockée dans la variable 'reponse'
nom = input("Quel est ton nom ? ")              # La fonction input() affiche la question et stocke la réponse textuelle de l'utilisateur dans 'nom'
age_texte = input("Quel est ton âge ? ")        # Stocke l'âge saisi (toujours du texte initialement) dans 'age_texte'
age = int(age_texte)                            # Le mot-clé int() convertit le texte de 'age_texte' en nombre entier et le stocke dans 'age'

B Les Conditions : La Logique if, elif, else

Ces mots-clés permettent au programme de prendre des décisions basées sur des comparaisons. [attachment_0](attachment)

Exemple Ligne par Ligne : 
age = 17                              # Initialise la variable 'age' à 17

if age >= 18:                         # Première condition : Si 'age' est supérieur ou égal à 18 (faux, car 17 < 18)
    print("Vous êtes majeur.")         # Cette ligne est ignorée car la condition est fausse
elif age >= 16:                       # Deuxième condition (vérifiée après l'échec du 'if') : Si 'age' est supérieur ou égal à 16 (vrai, car 17 >= 16)
    print("Vous pouvez conduire avec accompagnateur.") # Cette ligne est exécutée
else:                                 # Ce bloc est la condition par défaut (si toutes les autres sont fausses)
    print("Vous êtes mineur.")         # Cette ligne est ignorée car le 'elif' précédent était vrai

Structures de Données : Ranger l'Information

A Les Listes (Conteneurs Ordre et Modifiable)

Les listes sont des séquences d'éléments délimités par des crochets **`[]`**.

Exemple Ligne par Ligne : 
liste_courses = ["oeufs", "lait", "pain"]  # Crée une liste de chaînes de caractères (index 0, 1, 2)
print(liste_courses[0])                    # Affiche l'élément à l'index 0, soit "oeufs"
liste_courses.append("fromage")            # Ajoute l'élément "fromage" à la fin de la liste
liste_courses.remove("lait")               # Recherche et retire la première occurrence de la valeur "lait"

B Les Dictionnaires (Clé:Valeur)

Les dictionnaires sont délimités par des accolades **`{}`** et stockent les données sous forme de paires **Clé:Valeur**.

Exemple Ligne par Ligne : 
profil = {                           # Début du dictionnaire (accorde)
    "nom": "Alice",                  # Première paire: la clé "nom" pointe vers la valeur "Alice"
    "age": 28,                       # Deuxième paire: la clé "age" pointe vers la valeur 28
    "ville": "Paris"                 # Troisième paire
}                                    # Fin du dictionnaire
print(f"Nom: {profil['nom']}")     # Affiche "Nom: Alice" en accédant à la valeur via la clé "nom"
profil["age"] = 29                   # Modifie la valeur associée à la clé "age" et la met à 29

Le Moteur : Répétition et Réutilisabilité

A Les Boucles (for et while)

Les boucles servent à automatiser les tâches répétitives. [attachment_1](attachment)

Boucle FOR (Itération) :

Exemple Ligne par Ligne : 
fruits = ["pomme", "poire", "kiwi"]       # La liste des éléments à parcourir
for f in fruits:                          # Démarre la boucle FOR : pour chaque élément 'f' dans la liste 'fruits'
    print(f"J'aime la {f}")               # Exécute l'action (affichage) en utilisant la valeur courante de 'f'

Boucle WHILE (Conditionnelle) :

Exemple Ligne par Ligne : 
i = 5                                     # Initialise la variable de compteur 'i' à 5
while i > 0:                              # Démarre la boucle WHILE : Tant que 'i' est strictement supérieur à 0, répéter
    print(i)                              # Affiche la valeur courante de 'i'
    i -= 1                                # Décrémente 'i' de 1 (i = i - 1). C'est l'étape qui mène à la fin de la boucle.
# La boucle s'arrête quand i atteint 0

B Les Fonctions (def et return)

Une fonction est une mini-machine : vous lui donnez une entrée (argument), elle travaille, et vous donne une sortie (valeur de retour).

Exemple Ligne par Ligne : 
def calculer_aire_carre(cote):      # Le mot-clé 'def' définit la fonction. 'cote' est l'argument attendu.
    return cote * cote                 # Le mot-clé 'return' renvoie le résultat du calcul (cote * cote) à l'appelant de la fonction.

# L'appel à la fonction (avec la valeur 7 pour 'cote')
a = calculer_aire_carre(7)          # Appelle la fonction. La variable 'a' reçoit la valeur renvoyée (7 * 7 = 49).
print(a)                           # Affiche la valeur de 'a', soit 49.

Le Niveau Supérieur : Interfaces Graphiques (GUI) avec Tkinter

**Tkinter** est la bibliothèque standard qui transforme votre code Python en applications de bureau avec des fenêtres.

Code Décomposé Ligne par Ligne

import tkinter as tk                                 # 1. Importe la librairie Tkinter et lui donne le raccourci 'tk' pour plus de simplicité

# 2. Définit la fonction qui sera exécutée lors du clic sur le bouton
def saluer():                                        # Début de la définition de la fonction 'saluer'
    message_label.config(text="Bonjour ! Le message a changé !") # Utilise la méthode .config() pour modifier la propriété 'text' du widget 'message_label'

# 3. Initialisation de la Fenêtre Principale (le "root")
fenetre = tk.Tk()                                    # Crée l'objet Fenêtre principale (la boîte extérieure)
fenetre.title("Ma Première App")                     # Donne un titre à la fenêtre
fenetre.geometry("350x150")                          # Définit la taille initiale de la fenêtre (Largeur x Hauteur)

# 4. Créer et Placer les Widgets
# Widget de Texte (Label)
message_label = tk.Label(fenetre, text="Cliquez sur le bouton ci-dessous.") # Crée un Label (texte) dans la 'fenetre' avec un texte initial
message_label.pack(pady=10)                          # Place le Label. La méthode .pack() organise les widgets et pady ajoute un espacement vertical de 10 pixels

# Widget Bouton (Button)
bouton = tk.Button(                                 # Crée un Bouton dans la 'fenetre'
    fenetre,                                         # Le premier argument indique que le bouton appartient à la 'fenetre'
    text="Cliquez ici !",                           # Le texte affiché sur le bouton
    command=saluer,                                  # ATTENTION: Spécifie la fonction 'saluer' à appeler LORS du clic (SANS les parenthèses !)
    bg="#2563eb",                                    # Couleur de fond (code hexadécimal bleu)
    fg="white"                                       # Couleur du texte (blanc)
)
bouton.pack(pady=10)                                # Place le bouton avec un espacement

# 5. Démarrer la Boucle Principale
fenetre.mainloop()                                   # OBLIGATOIRE : Lance le moteur Tkinter. La fenêtre reste ouverte et attend les actions utilisateur.
**Points Clés Tkinter :**
  • .config() : Utilisé pour modifier dynamiquement les propriétés d'un widget (comme le texte d'un `Label`) APRES sa création.
  • command=saluer : On lie le clic du bouton à la fonction `saluer`. **Jamais de `()` ici**.
  • .mainloop() : Le moteur de l'application. Tant que cette boucle tourne, la fenêtre est visible et réactive.

Projet 1 : Jeu de Devinette (Console)

L'Objectif

Un jeu pour deviner un nombre secret. Il utilise import random, while True et les conditions if/elif/else.

Le Code Complet (Ligne par Ligne) :

import random                               # Importe la boîte à outils du hasard (module standard)

print("--- JEU DE DEVINETTE ---")           # Affiche le titre du jeu dans la console
nombre_secret = random.randint(1, 100)      # Utilise la fonction randint du module random pour choisir un nombre entre 1 et 100
essais = 0                                  # Initialise le compteur d'essais à zéro (c'est une variable)

print("Je pense à un nombre entre 1 et 100.") # Instructions pour l'utilisateur

while True:                                 # Démarre une boucle infinie (continue jusqu'à ce que 'break' soit atteint)
    # Tente d'exécuter le code suivant. Si une erreur de type (ValueError) survient, passe au bloc 'except'.
    try:                                    
        # Demande la proposition de l'utilisateur (texte) et tente de la convertir en entier (int)
        proposition = int(input("Ta proposition : ")) 
    except ValueError:
        print("Veuillez entrer un nombre valide.")  # Affiche un message d'erreur si la conversion en nombre a échoué
        continue                                    # Ignore le reste du code dans cette itération et recommence la boucle (while True)

    essais += 1                                 # Incrémente le compteur d'essais de 1 (raccourci pour essais = essais + 1)

    if proposition == nombre_secret:            # Condition de victoire : Si la proposition est strictement ÉGALE au nombre secret
        # Affiche le message de victoire, utilisant une f-string pour insérer la variable 'essais'
        print(f"Gagné ! Tu as trouvé en {essais} coups !") 
        break                                   # Force la sortie de la boucle 'while True' (fin du jeu)
    elif proposition < nombre_secret:           # Seconde condition (vérifiée si la première est fausse) : Si la proposition est PLUS PETITE que le secret
        print("C'est plus grand ⬆️")             # Donne un indice
    else:                                       # Condition par défaut (si proposition > nombre_secret)
        print("C'est plus petit ⬇️")             # Donne un autre indice
**Concepts Utilisés :**
  • random.randint(a, b) : Génère un entier aléatoire entre `a` et `b` (inclus).
  • while True : Crée une boucle qui s'exécutera indéfiniment jusqu'à ce que `break` soit rencontré.
  • try/except : Mécanisme de sécurité pour gérer les erreurs (ici, si l'utilisateur ne rentre pas un nombre).

Projet 2 : Calculatrice de Base (Console)

L'Objectif

Utiliser les **fonctions** pour organiser le code et le rendre modulaire.

Le Code Complet (Ligne par Ligne) :

# --- CALCULATRICE DE BASE ---

# 1. Définition des fonctions pour chaque opération (réutilisabilité)
def addition(x, y):             # Définit la fonction 'addition' qui prend deux arguments 'x' et 'y'
    return x + y                # Renvoie la somme de x et y

def soustraction(x, y):         # Définit la fonction 'soustraction'
    return x - y                # Renvoie la différence entre x et y

def multiplication(x, y):       # Définit la fonction 'multiplication'
    return x * y                # Renvoie le produit de x et y

def division(x, y):             # Définit la fonction 'division'
    if y == 0:                  # Vérifie si le diviseur 'y' est égal à zéro (division par zéro est impossible)
        return "Erreur: Division par zéro impossible." # Si vrai, renvoie un message d'erreur (texte/string)
    return x / y                # Sinon, renvoie le résultat de la division (nombre/float)

def calculer():                 # Définit la fonction principale qui gère la boucle du programme
    print("\n--- CALCULATRICE SIMPLE ---") # Affiche le titre
    print("Sélectionnez une opération:")   # Affiche le menu
    print("1. Addition")
    print("2. Soustraction")
    print("3. Multiplication")
    print("4. Division")
    print("5. Quitter")

    while True:                 # Boucle infinie pour permettre des calculs successifs
        choix = input("Entrez votre choix (1/2/3/4/5): ") # Demande le choix de l'opération

        if choix == '5':        # Si l'utilisateur choisit 5 (Quitter)
            print("Au revoir!") 
            break               # Sort de la boucle (fin du programme)

        # Vérifie si le choix est une des opérations valides
        if choix in ('1', '2', '3', '4'): 
            try:
                # Demande des deux nombres et conversion en float (pour gérer les décimales)
                num1 = float(input("Entrez le premier nombre: "))
                num2 = float(input("Entrez le deuxième nombre: "))
            except ValueError:
                print("Entrée invalide. Veuillez entrer des nombres.") # Gestion d'erreur si l'utilisateur n'entre pas de nombre
                continue                                            # Recommence la boucle

            # Aiguillage du programme vers la bonne fonction en fonction du choix
            if choix == '1':
                resultat = addition(num1, num2)     # Appelle la fonction 'addition'
            elif choix == '2':
                resultat = soustraction(num1, num2) # Appelle la fonction 'soustraction'
            elif choix == '3':
                resultat = multiplication(num1, num2)# Appelle la fonction 'multiplication'
            elif choix == '4':
                resultat = division(num1, num2)     # Appelle la fonction 'division'
                
                # Vérifie si le résultat de la division est le message d'erreur (type string)
                if isinstance(resultat, str): 
                    print(resultat)
                    continue                        # Recommence la boucle car l'opération a échoué

            print(f"Résultat: {resultat}")          # Affiche le résultat du calcul
        else:
            print("Choix invalide.")                # Message d'erreur si le choix n'est pas 1, 2, 3, 4 ou 5

# Appel de la fonction principale pour démarrer la calculatrice
calculer()
**Concepts Avancés :**
  • float() : Convertit une chaîne de caractères en nombre à virgule flottante (nombre décimal).
  • isinstance(objet, type) : Fonction qui vérifie si un objet est d'un type spécifique.

Vision par Ordinateur avec OpenCV

Qu'est-ce qu'OpenCV ?

**OpenCV** est la bibliothèque standard pour traiter les images et les vidéos. Elle permet au programme de "voir" en lisant et manipulant des pixels.

Installation (dans le Terminal)

pip install opencv-python # Installe le paquet 'opencv'

Code : Lire et Afficher une Image

import cv2                                  # 1. Importe la librairie OpenCV (souvent nommée 'cv2')
import os                                   # 2. Importe le module os pour gérer les chemins de fichiers

# NOTE: Assurez-vous d'avoir une image nommée 'photo.jpg' dans le même dossier que votre script.

chemin_image = 'photo.jpg'                  # Nom du fichier image à charger
if not os.path.exists(chemin_image):
    print("Erreur: Le fichier 'photo.jpg' n'a pas été trouvé.")
else:
    # 3. Lecture de l'image
    image = cv2.imread(chemin_image, cv2.IMREAD_COLOR) # Charge l'image depuis le chemin. IMREAD_COLOR = image en couleur.

    # 4. Affichage de l'image dans une nouvelle fenêtre
    cv2.imshow('Mon Image OpenCV', image)   # Crée une fenêtre nommée 'Mon Image OpenCV' et y affiche l'objet 'image'

    # 5. Attente d'une touche
    cv2.waitKey(0)                          # Le programme attend que l'utilisateur appuie sur n'importe quelle touche (0 = infini)

**Le Concept Clé (La Matrice) :**

Pour OpenCV, une image n'est qu'une immense **matrice NumPy** (un tableau de nombres). Chaque nombre représente l'intensité de couleur (Rouge, Vert, Bleu) d'un **pixel**.

Projet 3 : Carnet d'Adresses Simple (Console)

L'Objectif

Maîtriser l'imbrication des structures de données : une **Liste** (le carnet) contenant plusieurs **Dictionnaires** (chaque contact).

Le Code Complet (Ligne par Ligne) :

# --- CARNET D'ADRESSES (Liste de Dictionnaires) ---

# 1. La structure de données principale (Liste de Dictionnaires)
contacts = [                            # Début de la liste 'contacts'
    {                                   # Début du premier dictionnaire (contact)
        "nom": "Alice Dupont",          # Paire Clé:Valeur (Clé: nom, Valeur: Alice Dupont)
        "telephone": "0612345678",      # Deuxième paire
        "email": "alice@email.com"      # Troisième paire
    },                                  # Fin du dictionnaire
    # ... on pourrait ajouter d'autres dictionnaires ici ...
]                                       # Fin de la liste

def afficher_contacts(liste):           # Définit la fonction 'afficher_contacts', qui prend la 'liste' en argument
    """Affiche tous les contacts de la liste.""" # Docstring (description de la fonction)
    print("\n--- LISTE DES CONTACTS ---") # Affichage titre

    # La fonction enumerate() retourne à la fois l'index (i) et l'élément (contact) pendant l'itération
    for i, contact in enumerate(liste): 
        print(f"\nContact #{i + 1}")    # Affiche le numéro du contact (i + 1 car l'index 'i' commence à 0)
        
        # Accède aux valeurs du dictionnaire 'contact' par leur clé (entre crochets [])
        print(f"  Nom: {contact['nom']}") 
        print(f"  Tél: {contact['telephone']}")
        print(f"  Email: {contact['email']}")
    print("--------------------------\n") # Séparateur

# 2. Saisie des données du nouveau contact
nouveau_nom = input("Nom du nouveau contact : ")
nouveau_tel = input("Téléphone du nouveau contact : ")
nouveau_email = input("Email du nouveau contact : ")

nouveau_contact = {                     # Création d'un nouveau Dictionnaire avec les données saisies
    "nom": nouveau_nom,
    "telephone": nouveau_tel,
    "email": nouveau_email
}

contacts.append(nouveau_contact)        # Ajout du nouveau Dictionnaire à la Liste 'contacts'

# 3. Affichage après l'ajout
afficher_contacts(contacts)             # Appelle la fonction d'affichage avec la liste mise à jour
**Concept Clé : Imbrication**

Imaginez que le carnet soit un tiroir (`List`). Chaque contact est un classeur (`Dictionary`) dans le tiroir. Pour trouver le nom, vous cherchez la clé "nom" dans le classeur.

root.title("Convertisseur C°/F°")                   # Définit le titre de la fenêtre
root.geometry("300x150")                            # Définit la taille

# 3. Création et Placement des Widgets
# Label pour l'instruction
label_celsius = tk.Label(root, text="Température en Celsius (°C):") 
label_celsius.pack(pady=5)                          # Place le Label

# Champ de Saisie (Entry)
entry_celsius = tk.Entry(root)                      # Crée le champ de saisie
entry_celsius.pack(pady=5)                          # Place le champ de saisie

# Bouton de Conversion
convert_button = tk.Button(
    root, 
    text="Convertir", 
    command=celsius_vers_fahrenheit,                # Lie le clic du bouton à la fonction de conversion (SANS parenthèses)
    bg="#2563eb", 
    fg="white"
)
convert_button.pack(pady=10)                        # Place le bouton

# Label pour le Résultat
result_label = tk.Label(root, text="Résultat: -")
result_label.pack(pady=5)                           # Place le Label de résultat

# 4. Lancement de la boucle principale
root.mainloop()                                     # Démarre le moteur graphique Tkinter (la fenêtre apparaît et attend les interactions)
**Fonctions et Widgets :**
  • tk.Tk() : La classe de base pour toutes les applications Tkinter.
  • tk.Entry() : Un champ de texte pour que l'utilisateur puisse saisir des données.
  • .get() : La méthode essentielle pour extraire le texte saisi dans un `Entry`.
